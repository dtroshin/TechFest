2.4 - An Applet With Graphics

1) The Graphics class is located in the java.awt package.

2) Some useful methods used for drawing are:

  • drawOval(int x, int y, int width, int height);
        //This Draws the outline of an oval. If we want a filled oval replace "draw" with "fill"

  • drawPolygon(int[] xPoints, int[] yPoints, int nPoints)
        //Draws a closed polygon defined by arrays of x and y coordinates. If we want a filled polygon replace "draw" with "fill"

  • drawString(String str, int x, int y)
        //Draws the text given by the specified string, using this graphics context's current font and color.

  • drawLine(int x1, int y1, int x2, int y2)
        //Draws a line, using the current color, between the points (x1, y1) and (x2, y2) in this graphics context's coordinate system.
