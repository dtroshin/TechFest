2.2 - A Trivial Applet Subclass

1) The Applet class is located in the java.applet package
The Color class is located in the java.awt package

2) The init() method is located in the java.applet package and can be found in the Applet Class
it is Called by the browser or applet viewer to inform this applet that it has been loaded into the system.

The paint() method is located in the java.awt package and can be found in the Container Class
Paints the container.

3) The Color method is located in the java.awt package
some fields include black, blue, cyan, dark_gray, green, and many more.

One way to create our own colors is by creating new Color objects through constructors.
i.e. Color myColor = new Color(int r, int g, int b);
